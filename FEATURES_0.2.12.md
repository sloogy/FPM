# v0.2.12 – Rotation-Priorität: leere Füller zuerst

- Leere Füller werden wieder zuverlässig vorgeschlagen.
- Farbspektrum ist kein harter Filter mehr, sondern Score-Bonus/Malus.

  1. leere Füller
  2. lange nicht genutzt

  4. Feder-/Tintenmatch
  5. Reinigungssicherheit
  6. Beliebtheit
- Hinweise zeigen jetzt z. B. „leer“, „Tage nicht genutzt“, „Beliebtheit 4/5“.
