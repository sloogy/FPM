# v0.2.15 – Kopieren, Refill und Dublettenprüfung

- Tinten können als neues Fass / Refill kopiert werden.
- Beim Refill werden Eigenschaften übernommen, Preis/Kaufdatum/Fassgröße können neu gesetzt werden.
- Dublettenprüfung für Tinten: Marke + Name + Flaschengröße + aktiv/nicht leer.
- Füller können als neues Exemplar kopiert werden.
- Beim Füller-Kopieren werden Preise/Werte neu eingetragen und der Kaufpreis wieder in den Ausgaben-Tracker synchronisiert.
- Dublettenprüfung für Füller: Marke + Modell + Farbe + Füllgröße.
